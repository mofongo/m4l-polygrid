# polygrid

An 8-track polymeter step sequencer for Ableton Live — a Max for Live MIDI effect.

Every track is its own loop with an independent step count, length, and phase offset,
cycling against a one-bar window locked to Live's transport.

**[Landing page & download →](https://mofongo.github.io/m4l-polygrid/)**

Or grab [`polygrid.amxd`](polygrid.amxd) directly and drop it on a MIDI track before any instrument.

The `.amxd` is frozen (self-contained) — one file is all you need.
